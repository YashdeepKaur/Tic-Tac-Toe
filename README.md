# Tic Tac Toe (Java Swing)

A two-player Tic Tac Toe game with a dark-themed Swing GUI.

## How to Run

```bash
javac TicTacToe.java
java TicTacToe
```

## Features
- Custom painted board and pieces
- Win detection with highlighted line
- Score tracking
